# day 43 texture basics

today is about **textures** in three.js.

textures are images mapped onto geometry.
understanding how they load and map is essential.

goal:
- load textures with `TextureLoader`
- apply textures to materials
- understand UVs and wrapping
- understand texture filtering

## runnable demo

a demo exists in `days/day-043/demo`.

with vite:
- `npm run dev`
- open `/days/day-043/demo/index.html`
