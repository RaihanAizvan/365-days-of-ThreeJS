# day 61 shadow types and softness

today is about **shadow types and softness**.

three.js supports multiple shadow map types and softness options.

the choice affects quality and performance.

key topics:
- BasicShadowMap
- PCFShadowMap
- PCFSoftShadowMap
- VSMShadowMap
- shadow radius and filtering

goal:
- understand shadow map types
- choose the right shadow filter
- balance quality and performance
