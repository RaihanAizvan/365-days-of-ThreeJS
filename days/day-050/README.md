# day 50 anisotropy theory

today is about **anisotropic filtering**.

anisotropy improves texture quality at grazing angles.
it reduces blur and shimmering on surfaces like floors and roads.

goal:
- understand what anisotropy does
- know how to set `texture.anisotropy`
- learn how to query max anisotropy from the renderer

## runnable demo

a demo exists in `days/day-050/demo`.
