# day 46 uv mapping

today is about **UV mapping**.

UVs define how a 2D texture is mapped onto a 3D surface.

goal:
- understand uv coordinates (0..1)
- learn how textures repeat and wrap in UV space
- inspect and modify uv attributes

## runnable demo

a demo exists in `days/day-046/demo`.

with vite:
- `npm run dev`
- open `/days/day-046/demo/index.html`
