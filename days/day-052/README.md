# day 52 mirror wrapping

today is about **mirror wrapping**.

mirror wrapping alternates a texture by mirroring each repeat tile.

wrap modes:
- ClampToEdgeWrapping (default)
- RepeatWrapping
- MirroredRepeatWrapping

goal:
- understand how mirrored repeat works
- compare mirror vs repeat vs clamp
- use repeat/offset with mirrored repeat

## runnable demo

a demo exists in `days/day-052/demo`.

with vite:
- `npm run dev`
- open `/days/day-052/demo/index.html`
