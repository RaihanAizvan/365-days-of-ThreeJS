# day 38 geometry attributes

today is about **geometry attributes**.

attributes live on `BufferGeometry` and define the data sent to the GPU.

goal:
- inspect geometry attributes
- understand attribute counts
- update attributes and mark `needsUpdate`
- know common attributes (`position`, `normal`, `uv`, `color`)

## runnable demo

a demo exists in `days/day-038/demo`.

with vite:
- `npm run dev`
