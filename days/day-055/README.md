# day 55 material basics

today is about **material basics**.

materials define how surfaces interact with light.

three.js provides many material types for different rendering needs.

key concepts:
- MeshBasicMaterial (unlit)
- MeshStandardMaterial (PBR)
- Material properties

goal:
- understand material types
- learn basic material properties
- choose the right material for the job

a demo exists in `days/day-055/demo`.
