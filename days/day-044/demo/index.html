📊 Image Format Demo

PNG

Lossless compression with full transparency support

Format Size (512×512): ~512 KB
Compression: Lossless
Transparency: Full RGBA
Use Case: Graphics

Format Comparison (512×512)

PNG 512 KB, Lossless
JPEG ~85 KB, Lossy
WebP ~70 KB, Lossy/Lossless
HDR ~200 KB, Float32
💡 Tip: Choose formats based on content type. Photos? JPEG/WebP. Graphics? PNG. HDR lighting? EXR/HDR.
