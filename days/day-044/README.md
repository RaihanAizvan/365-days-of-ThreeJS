# day 44 image formats

today is about **image formats** in three.js and web graphics.

different image formats have tradeoffs in compression, quality, and browser support.
understanding when to use each format is crucial for performance.

goal:
- learn PNG (lossless, transparency support)
- learn JPEG (lossy, good for photographs)
- learn WebP (modern, better compression)
- learn EXR and HDR (high dynamic range)
- understand format selection criteria

## runnable demo

a demo exists in `days/day-044/demo`.

with vite:
- `npm run dev`
- open `/days/day-044/demo/index.html`
