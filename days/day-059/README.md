# day 59 shadows basics

today is about **shadows basics**.

shadows add realism and depth to 3D scenes.

three.js supports real-time shadow mapping.

key concepts:
- enabling shadows on renderer
- lights that cast shadows
- objects that cast/receive shadows
- shadow map configuration

goal:
- understand shadow system
- configure shadow quality
- optimize shadow performance
