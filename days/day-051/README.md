# day 51 repeat wrapping

today is about **repeat wrapping**.

textures can repeat (tile) across a surface.
this is controlled by wrap modes and the `repeat` vector.

goal:
- understand wrapS / wrapT
- use RepeatWrapping
- tile textures with `repeat`
- use offset for scrolling textures

## runnable demo

a demo exists in `days/day-051/demo`.

with vite:
