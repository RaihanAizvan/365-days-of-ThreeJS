# day 58 lighting basics

today is about **lighting basics**.

lights are essential for lit materials to be visible.

three.js provides several light types for different effects.

key light types:
- AmbientLight
- DirectionalLight
- PointLight
- SpotLight

goal:
- understand different light types
- learn when to use each light
- create effective lighting setups

## runnable demo

a demo exists in `days/day-058/demo`.

with vite:
- `npm run dev`
- open `/days/day-058/demo/index.html`
