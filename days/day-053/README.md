# day 53 texture transformations

today is about **texture transformations**.

texture transformations allow you to offset, rotate, and center textures on geometry.

key properties:
- texture.offset (Vector2)
- texture.rotation (number in radians)
- texture.center (Vector2, rotation pivot point)

goal:
- understand texture offset
- understand texture rotation
- understand rotation center point

a demo exists in `days/day-053/demo`.
